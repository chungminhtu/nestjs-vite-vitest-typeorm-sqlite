# notice

This project is to demo capable of running Nestjs with Vite (SWC) and Vitest for fastest Dev experience.

- The package.json has to setup `"type": "module"` to build as ESM package and the project build with `Vite build` command will output as `cjs` file not `js` file.
- We can run this `cjs` file normally with Node or Nodemon.
